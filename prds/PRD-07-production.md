## Objective

Harden SLAM-COKO for repeatable deployment with artifact validation, bandwidth budgeting, observability, and release gates around the paper-critical behaviors.

## Context (from paper)

The paper’s value proposition depends on communication efficiency and robust unknown-pose map fusion. Productionization must keep those properties visible and testable.

**Paper reference:** §1, §4, Tables 3-5, Conclusion

- [ ] Artifact bundle contains merged map, metrics JSON, bandwidth JSON, and gap report
- [ ] Failure policies distinguish loop-detection miss, registration miss, and PGO instability
- [ ] Test: `uv run pytest tests/production/test_release_checks.py -v` passes
